HOW TO INSTALL ON GITHUB

Option A - Replace files directly on GitHub:
1. Open your Conversia MX website repository on GitHub.
2. Upload these files to the same folder where your current index.html is located.
3. Replace the existing index.html with this new index.html.
4. Make sure styles.css is uploaded in the same root folder.
5. Keep the CNAME file if you are using conversiamx.com as the custom domain.
6. Commit the changes.
7. Wait 1-5 minutes, then open https://conversiamx.com/.

Option B - Using GitHub Desktop:
1. Clone/open your website repository in GitHub Desktop.
2. Copy all files from this folder into the repository folder.
3. Replace the old index.html.
4. Commit the changes with a message like: Update Conversia MX website pages.
5. Push to GitHub.
6. Wait for GitHub Pages to redeploy.
